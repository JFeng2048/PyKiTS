Jinja2控制结构练习

1. 条件语句 (if-elif-else)

根据用户状态显示不同信息：

{% for user in users %} {% endfor %}
用户名 年龄 状态 角色 描述
{{ user.name }} {{ user.age }} {{ '活跃' if user.is_active else '非活跃' }} {{ user.role }} {% if user.role == 'admin' %} 管理员用户 {% elif user.role == 'manager' %} 经理用户 {% elif user.role == 'user' %} 普通用户 {% else %} 未知角色 {% endif %} {% if user.age > 35 %} (经验丰富) {% elif user.age < 25 %} (初级) {% endif %}

2. 循环控制 (for循环)

产品列表（带库存状态）：

{% for product in products %} {% endfor %}
# 产品名称 价格 库存 类别 库存状态
{{ loop.index }} {{ product.name }} ¥{{ product.price }} {{ product.stock }} {{ product.category }} {% if product.stock < 20 %} 库存不足 {% elif product.stock < 50 %} 库存中等 {% else %} 库存充足 {% endif %}

循环信息：

  • 产品总数：{{ products|length }}
  • 第一个产品：{{ products[0].name if products else '无' }}
  • 最后一个产品：{{ products[-1].name if products else '无' }}

3. 嵌套循环

类别和产品关系：

{% for category, items in categories.items() %}

{{ category }}

    {% for item in items %}
  • {{ item }} {% set found = false %} {% for product in products %} {% if product.name == item %} - ¥{{ product.price }} {% set found = true %} {% break %} {% endif %} {% endfor %} {% if not found %} - 暂无价格信息 {% endif %}
    • {% endfor %}
{% endfor %}

4. 循环变量的使用

用户列表（带循环信息）：

    {% for user in users %}
  • {% if loop.first %} [第一个] {% elif loop.last %} [最后一个] {% endif %} {{ loop.index }}. {{ user.name }} ({{ user.role }}) {% if loop.index0 % 2 == 0 %} (奇数位置) {% endif %}
    • {% endfor %}

循环统计：

总共有 {{ loop.length }} 个用户（如果在循环内可用）

5. 复杂条件和循环组合

特殊用户展示：

    {% for user in users %} {% if user.is_active and (user.role == 'admin' or user.age > 30) %}
  • {{ user.name }} - 重要用户 ({{ user.role }}, {{ user.age }}岁)
    • {% endif %} {% endfor %}

价格区间产品：

    {% for product in products %} {% if product.price > 1000 and product.price < 5000 %}
  • {{ product.name }} - ¥{{ product.price }} ({{ product.category }})
    • {% endif %} {% endfor %}
