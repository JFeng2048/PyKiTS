Jinja2基础语法示例

1. 变量显示

Hello, {{ name }}!

Your age is {{ age }}.

{{ '{{ name }}' }} 和 {{ '{{ age }}' }} 是变量

2. 表达式

2 + 3 = {{ 2 + 3 }}

Is 5 greater than 3? {{ 5 > 3 }}

Your age next year: {{ age + 1 }}

{{ '{{ 2 + 3 }}' }}、{{ '{{ 5 > 3 }}' }} 和 {{ '{{ age + 1 }}' }} 是表达式

3. 字符串操作

Your name in uppercase: {{ name|upper }}

Your name in lowercase: {{ name|lower }}

{{ '{{ name|upper }}' }} 使用了过滤器

4. 列表访问

First hobby: {{ hobbies[0] }}

Second hobby: {{ hobbies[1] }}

{{ '{{ hobbies[0] }}' }} 访问列表元素

5. 字典访问

Username: {{ user.username }}

Email: {{ user.email }}

{{ '{{ user.username }}' }} 访问字典属性

6. 注释示例

This is visible content.

{# This is a comment and won't be visible in the output #}

{{ '{# This is a comment #}' }} 是Jinja2注释

7. 转义与安全输出

转义后: {{ html_content }}

原始输出: {{ html_content|safe }}

{{ '{{ html_content|safe }}' }} 禁用了自动转义

8. 日期和时间

Current date: {{ current_date.strftime('%Y-%m-%d') }}

Current time: {{ current_date.strftime('%H:%M:%S') }}

使用Python的strftime方法格式化日期

9. 默认值

Optional value: {{ optional_value|default('Not provided') }}

{{ '{{ optional_value|default(\'Not provided\') }}' }} 设置了默认值

10. 测试值是否存在

{% if missing_value is defined %}

Missing value exists: {{ missing_value }}

{% else %}

Missing value is not defined.

{% endif %}

使用{% raw %}{% if missing_value is defined %}{% endraw %} 测试变量是否存在
