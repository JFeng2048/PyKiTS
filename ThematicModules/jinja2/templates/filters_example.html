Jinja2变量和过滤器示例

1. 基本变量访问

点表示法访问字典属性:

用户名: {{ user.username }}

邮箱: {{ user.email }}

{{ '{{ user.username }}' }} 和 {{ '{{ user.email }}' }}

下标表示法访问列表元素:

第一个爱好: {{ hobbies[0] }}

第二个爱好: {{ hobbies[1] }}

{{ '{{ hobbies[0] }}' }} 和 {{ '{{ hobbies[1] }}' }}

2. 字符串过滤器

过滤器 示例 结果 代码
upper 将字符串转换为大写 {{ message|upper }} {{ '{{ message|upper }}' }}
lower 将字符串转换为小写 {{ message|lower }} {{ '{{ message|lower }}' }}
capitalize 首字母大写 {{ message|capitalize }} {{ '{{ message|capitalize }}' }}
title 每个单词首字母大写 {{ message|title }} {{ '{{ message|title }}' }}
trim 去除首尾空白 {{ spaced_message|trim }} {{ '{{ spaced_message|trim }}' }}
length 获取字符串长度 {{ message|length }} {{ '{{ message|length }}' }}
replace 替换子串 {{ message|replace('world', 'Jinja2') }} {{ '{{ message|replace(\'world\', \'Jinja2\') }}' }}
default 变量未定义时使用默认值 {{ undefined_var|default('变量未定义') }} {{ '{{ undefined_var|default(\'变量未定义\') }}' }}
safe 输出原始HTML {{ html_content|safe }} {{ '{{ html_content|safe }}' }}

3. 数字过滤器

过滤器 示例 结果 代码
abs 绝对值 {{ negative_number|abs }} {{ '{{ negative_number|abs }}' }}
round 四舍五入(保留2位小数) {{ pi|round(2) }} {{ '{{ pi|round(2) }}' }}
int 转换为整数 {{ string_number|int }} {{ '{{ string_number|int }}' }}
float 转换为浮点数 {{ string_float|float }} {{ '{{ string_float|float }}' }}

4. 列表过滤器

过滤器 示例 结果 代码
first 第一个元素 {{ numbers|first }} {{ '{{ numbers|first }}' }}
last 最后一个元素 {{ numbers|last }} {{ '{{ numbers|last }}' }}
length 列表长度 {{ numbers|length }} {{ '{{ numbers|length }}' }}
sort 排序 {{ unsorted_numbers|sort }} {{ '{{ unsorted_numbers|sort }}' }}
join 连接成字符串 {{ hobbies|join(', ') }} {{ '{{ hobbies|join(\', \') }}' }}

5. 字典过滤器

过滤器 示例 结果 代码
length 字典长度 {{ user|length }} {{ '{{ user|length }}' }}
dictsort 按键排序 {{ unsorted_dict|dictsort }} {{ '{{ unsorted_dict|dictsort }}' }}

6. 日期和时间过滤器

过滤器 示例 结果 代码
date 格式化日期 {{ now|date('Y-m-d') }} {{ '{{ now|date(\'Y-m-d\') }}' }}
time 格式化时间 {{ now|time('H:i:s') }} {{ '{{ now|time(\'H:i:s\') }}' }}
datetime 格式化日期时间 {{ now|datetime('Y-m-d H:i:s') }} {{ '{{ now|datetime(\'Y-m-d H:i:s\') }}' }}

7. 过滤器链式调用

多个过滤器依次应用:

原始字符串: "{{ raw_string }}"

应用过滤器后: "{{ raw_string|trim|title|replace('World', 'Everyone') }}"

{{ '{{ raw_string|trim|title|replace(\'World\', \'Everyone\') }}' }}
