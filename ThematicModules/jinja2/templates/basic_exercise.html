2. 列表操作

你的技能列表:

    {% for skill in skills %}
  • {{ loop.index }}. {{ skill }}
    • {% endfor %}
{% if skills|length > 0 %}

{% else %}

请添加你的技能

{% endif %}

4. 表达式和过滤器

明年你将 {{ age + 1 }} 岁

你的名字（大写）: {{ name|upper }}

技能列表（逗号分隔）: {{ skills|join(', ') }}

当前日期（格式化）: {{ current_time|string }}
