Jinja2扩展功能示例

扩展功能概述

Jinja2的扩展系统允许开发者扩展模板语言的功能，添加新的标签、过滤器、测试和全局函数。本示例展示了多个自定义扩展的使用方法。

当前启用的扩展：

  • 文本处理扩展：提供文本截断、高亮、slugify等功能
  • 重复标签扩展：提供{% repeat %}标签用于循环执行代码块
  • 缓存块扩展：提供{% cache %}标签用于缓存模板块
  • 调试扩展：提供调试工具和信息显示功能

1. 文本处理扩展

文本截断

原始文本：{{ long_text }}

截断到10个词：{{ long_text | truncate_words(10) }}

截断到15个词并添加自定义后缀：{{ long_text | truncate_words(15, suffix='... [继续阅读]') }}

文本高亮

在文本中高亮'Jinja2'和'template'：

{{ long_text | highlight(['Jinja2', 'template']) }}

生成Slug

标题：{{ page_title }}

生成的Slug：{{ page_title | slugify }}

自定义分隔符的Slug：{{ page_title | slugify('_') }}

首字母大写

原始文本：{{ "hello world of templates" }}

首字母大写：{{ "hello world of templates" | capitalize_words }}

2. 重复标签扩展

基础重复功能

生成5个按钮：

{% repeat 5 %} {% endrepeat %}

使用循环变量

带循环信息的重复块：

{% repeat 4 %}
项目 {{ loop.index }}/{{ loop.length }} - {% if loop.first %}第一个{% elif loop.last %}最后一个{% else %}中间的{% endif %}
{% endrepeat %}

动态重复次数

根据变量动态生成元素：

{% repeat item_count %} 动态项目 {{ loop.index }} {% endrepeat %}

3. 缓存块扩展

基础缓存功能

这个内容块被缓存：

{% cache 'sample_block' %}
缓存生成时间：{{ current_time }}

这里是一些可能需要缓存的内容，例如从数据库获取的数据或复杂的计算结果。

缓存键：sample_block

{% endcache %}

带超时的缓存

这个内容块被缓存，但有300秒的超时时间：

{% cache 'time_sensitive_block', 300 %}
时间敏感缓存生成时间：{{ current_time }} （300秒后过期）

这里是一些需要定期更新的内容，例如新闻摘要或统计数据。

{% endcache %}

带动态键的缓存

使用动态键缓存不同用户的内容：

{% cache 'user_specific_block_' ~ user_id %}
用户特定缓存生成时间：{{ current_time }}

为用户 {{ user_id }} 定制的内容。

{% endcache %}

4. 调试扩展

调试块

使用调试块查看渲染时间：

{% debug 'test_block' %}

这是一个简单的段落内容。

调试块会自动记录渲染时间和错误信息。

{% enddebug %}

变量转储

转储产品列表变量的信息：

{% dump products %}

性能计时

测量复杂操作的执行时间：

{% timeit 'complex_operation' %} {% for product in products %} {% endfor %}
ID 名称 价格 库存
{{ product.id }} {{ product.name }} {{ product.price }} {{ product.stock }}
{% endtimeit %}

调试过滤器

使用调试过滤器检查变量类型：

products变量的类型：{{ products | type_of }}

第一个产品的类型：{{ products[0] | type_of }}

5. 综合应用示例

产品列表页面

{% cache 'product_list_' ~ page_num, 600 %}

产品列表（第 {{ page_num }} 页）

{% debug 'product_table' %} {% for product in products %} {% endfor %}
ID 名称 描述 价格 库存状态
{{ product.id }} {{ product.name }} {{ product.description | truncate_words(10) }} {{ product.price }} {% if product.stock > 0 %} 有货 {% else %} 缺货 {% endif %}
{% enddebug %}

分页导航：

{% repeat total_pages %} {% if loop.index == page_num %} {{ loop.index }} {% else %} {{ loop.index }} {% endif %} {% endrepeat %}
{% endcache %}
