Jinja2控制结构示例

1. 基本条件语句

if语句

{% raw %}{% if user.is_logged_in %} <p>Welcome back, {{ user.username }}!</p> {% endif %}{% endraw %}

渲染结果:

{% if user.is_logged_in %}

Welcome back, {{ user.username }}!

{% endif %}

if-else语句

{% raw %}{% if user.is_admin %} <p>You are an administrator.</p> {% else %} <p>You are a regular user.</p> {% endif %}{% endraw %}

渲染结果:

{% if user.is_admin %}

You are an administrator.

{% else %}

You are a regular user.

{% endif %}

if-elif-else语句

{% raw %}{% if score >= 90 %} <p>优秀</p> {% elif score >= 80 %} <p>良好</p> {% elif score >= 60 %} <p>及格</p> {% else %} <p>不及格</p> {% endif %}{% endraw %}

渲染结果 (当前分数: {{ score }}):

{% if score >= 90 %}

优秀

{% elif score >= 80 %}

良好

{% elif score >= 60 %}

及格

{% else %}

不及格

{% endif %}

2. 条件运算符

逻辑运算符 (and, or, not)

{% raw %}{% if user.age >= 18 and user.country == 'China' %} <p>You are eligible.</p> {% endif %}{% endraw %}

渲染结果:

{% if user.age >= 18 and user.country == 'China' %}

You are eligible.

{% endif %}

成员运算符 (in, not in)

{% raw %}{% if 'admin' in user.roles %} <p>You have admin privileges.</p> {% endif %}{% endraw %}

渲染结果:

{% if 'admin' in user.roles %}

You have admin privileges.

{% endif %}

测试运算符 (is, is not)

{% raw %}{% if variable is defined %} <p>The variable exists: {{ variable }}</p> {% else %} <p>The variable is not defined.</p> {% endif %}{% endraw %}

渲染结果:

{% if variable is defined %}

The variable exists: {{ variable }}

{% else %}

The variable is not defined.

{% endif %}

3. 基本循环

遍历列表

{% raw %}<ul> {% for item in items %} <li>{{ item }}</li> {% endfor %} </ul>{% endraw %}

渲染结果:

    {% for item in items %}
  • {{ item }}
    • {% endfor %}

遍历字典

{% raw %}<ul> {% for key, value in user.items() %} <li>{{ key }}: {{ value }}</li> {% endfor %} </ul>{% endraw %}

渲染结果:

    {% for key, value in user.items() %}
  • {{ key }}: {{ value }}
    • {% endfor %}

4. 循环索引和属性

{% raw %}<ul> {% for item in items %} <li> {{ loop.index }}. {{ item }} {% if loop.first %} <span class="highlight">(First)</span>{% endif %} {% if loop.last %} <span class="highlight">(Last)</span>{% endif %} </li> {% endfor %} </ul>{% endraw %}

渲染结果:

    {% for item in items %}
  • {{ loop.index }}. {{ item }} {% if loop.first %} (First){% endif %} {% if loop.last %} (Last){% endif %}
    • {% endfor %}

循环信息:

  • 总项目数: {{ items|length }}
  • 循环变量可用属性: loop.index, loop.index0, loop.revindex, loop.revindex0, loop.first, loop.last, loop.length

5. 循环控制

else子句

{% raw %}<ul> {% for item in empty_list %} <li>{{ item }}</li> {% else %} <li>No items found.</li> {% endfor %} </ul>{% endraw %}

渲染结果:

    {% for item in empty_list %}
  • {{ item }}
    • {% else %}
  • No items found.
    • {% endfor %}

break和continue

{% raw %}<ul> {% for item in items_with_control %} {% if item == 'stop' %} {% break %} {% endif %} {% if item == 'skip' %} {% continue %} {% endif %} <li>{{ item }}</li> {% endfor %} </ul>{% endraw %}

渲染结果 (遇到'stop'停止，遇到'skip'跳过):

    {% for item in items_with_control %} {% if item == 'stop' %} {% break %} {% endif %} {% if item == 'skip' %} {% continue %} {% endif %}
  • {{ item }}
    • {% endfor %}

6. 嵌套循环

{% raw %}<table> {% for row in table_data %} <tr> {% for cell in row %} <td>{{ cell }}</td> {% endfor %} </tr> {% endfor %} </table>{% endraw %}

渲染结果:

{% for row in table_data %} {% for cell in row %} {% endfor %} {% endfor %}
{{ cell }}

7. 循环和条件结合

{% raw %}<ul> {% for user in users %} {% if user.is_active %} <li class="active">{{ user.username }} (Active)</li> {% else %} <li class="inactive">{{ user.username }} (Inactive)</li> {% endif %} {% endfor %} </ul>{% endraw %}

渲染结果 (根据用户状态显示不同样式):

    {% for user in users %} {% if user.is_active %}
  • {{ user.username }} (Active)
    • {% else %}
  • {{ user.username }} (Inactive)
    • {% endif %} {% endfor %}
