    Jinja2模板继承练习

    这个页面演示了Jinja2的模板继承功能。本页面是一个子模板，继承自基础模板base_exercise.html。

    提示：在子模板中，我们可以通过{{ '{{ super() }}' }}函数来调用父模板中定义的内容，同时添加自己的内容。

    1. 模板继承的基本概念

    模板继承允许你创建一个基础模板（父模板），定义网站的基本结构，然后在子模板中覆盖特定的部分。

    {# 基础模板示例 #} {% raw %}{% block content %} {% endblock %}{% endraw %}
    {# 子模板示例 #} {% raw %}{% extends "base.html" %} {% block content %} {{ super() }} {% endblock %}{% endraw %}

    2. 如何使用块（Blocks）

    块是模板中可以被子模板覆盖的部分。在基础模板中，你可以使用{% raw %}{% block 名称 %}{% endblock %}{% endraw %}来定义块。

    常见的块包括：

    • head - 页面头部信息
    • title - 页面标题
    • styles - CSS样式
    • scripts - JavaScript脚本
    • content - 主要内容区域
    • footer - 页脚信息
